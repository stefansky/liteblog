## 4. 组件通信

### 4.1 通信方式
- props/$emit：父子组件通信
- $parent/$children：直接访问实例
- provide/inject：跨层级通信
- $attrs/$listeners：封装组件传参
- EventBus（非官方推荐）
- Vuex：状态集中管理


### 4.2 最佳实践
- 通信方式选择
- 性能优化
- 代码组织
- 优先使用 props / emit
- 多层建议用 provide/inject 或 Vuex
- 避免滥用 $parent / $children
